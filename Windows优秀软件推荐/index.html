Windows 优秀软件推荐

本文最后更新于 2023年10月18日 中午

想法来自于 酷软清单，用于记录自己曾经使用或者正在使用觉得还不错的 Windows 上的软件

本清单适用于

  • Windows 个人用户
  • 寻找解决问题的工具
  • 提高效率和生产力
  • 美化与整理方案

※：适合自己的才是最好的

软件入选规则

  • 有着良好的使用体验，包括但不限于

    • 专注核心功能的完善和加强
    • 干扰内容（广告、通知等）少
    • 界面优美
    • 免费或者合理的价格

  • 使用人群具有特定性的软件将不被选入，尽管它们足够好。例如：Cheat Engine、WinSCP

符号描述规则

  • 💻该产品的开发者或者开发项目
  • 🎈该产品的基本功能
  • ✨该产品的特点功能
  • ⭕值得注意的地方
  • 💖个人的使用感觉

基础应用

解压缩

  • 7-Zip
    • 🎈开源免费的解压缩工具
    • ✨支持一些压缩率很高的压缩格式
    • ⭕压缩速度偏慢
    • ⭕不能查看 rar 的注释内容，不支持使用 rar 档内的恢复记录解压损坏的 rar
      • 建议需要处理 rar 格式压缩包的用户使用 WinRAR
    • 💻GitHub 项目：M2Team/NanaZip。7-Zip 的分叉版本，特点是适配了 Windows 11 的右键菜单

检索启动

  • Everything

    • 🎈轻量化的文件名搜索工具
    • ⭕适合对文件进行批量整理时使用
    • ⭕对绝大部分用户而言，建议使用 Everything 代替 Windows Search。后者可在 Windows Service 中关闭

  • Listary

    • 🎈轻量化的文件名启动工具
    • ⭕很多人将其与 Everything 进行比较，但两者事实上的目标就不一致：前者在于文件查找，后者在于文件启动
    • ⭕Listary 的核心是：增强对话框。例如，使用 ctrl + G 快速切换到当前目录

浏览器

  • Edge

    • 🎈新版 Edge 浏览器使用 Chromium 内核，与 Windows 融为一体
    • ✨出奇的省电
    • ⭕有很不好用的安卓客户端

  • Firefox

    • 🎈使用了 Firefox Quantum 后速度大幅提升，定制功能强大

    • ✨有非常好用的安卓客户端

    • ✨有许多优秀的修改版本，例如 - Waterfox有许多优秀的修改版本，例如 WaterFox（典型的玩梗）

    • ⭕尽量使用国际版，可以在 Firefox FTP 下载

    • ⭕国际账号无法同步，请使用国内同步

  • Chrome

    • 🎈其内核被许多知名浏览器使用，拥有非常丰富的扩展程序
    • ✨近乎垄断了浏览器领域
    • ⭕国内账号同步存在问题
    • ⭕有中规中矩的安卓客户端（不能使用扩展）

录屏工具

  • OBS Studio
    • 🎈💻GitHub 项目：obsproject/obs-studio
    • ⭕视频录制和实时推流直播软件的首选
    • ✨有了视频采集卡，可以通过 OBS 将 Switch 等设备投屏到电脑

输入法

  • 微软拼音输入法

    • 🎈Windows 自带输入法

    • ⭕简洁、好用

    • ✨双拼自带支持自然码、微软双拼和智能 ABC，可以通过导入如下的 *.reg 文件到注册表从而开启小鹤双拼

  • AutoHotkey

    • 🎈热键脚本语言，功能极其强大，绝不仅限于输入法。但我而言的主要用途在于键位映射
    • 💖我主要通过它实现了两个功能：CapsLock 轻按切换中英文，长按大小写锁定；右 ALT 配合 WSADQE 进行上下左右以及翻页切换

  • 💖 窗口置顶：

系统美化

  • Auto Dark Mode

    • 🎈让系统自动在黑暗模式和明亮模式之间切换
    • ⭕微软商店中的 UWP 版本

  • Coodesker

    • 🎈体积极小的桌面整理工具，类似于 Fences
    • ⭕似乎是某个开发者的小作品，能够查询到的信息很少

下载工具

  • Motrix

    • 🎈基于 aria2的下载器
    • ✨简洁明了的图形操作界面，功能齐全
    • ✨支持BT和磁力链任务，并且每天自动更新 Tracker 服务器列表
    • ⭕目前该软件似乎已经停止维护

  • qBittorrent Enhanced Edition

    • 💻GitHub 项目：c0re100/qBittorrent-Enhanced-Edition
    • 🎈基于 qBittorrent 开发出来的一个功能增强分支
    • ✨可以自动屏蔽迅雷等吸血 BT 下载工具
    • ✨自动更新订阅的 Tracker 服务器列表
    • ⭕很多 PT 站并不支持二次开发的 BT 下载工具

音乐

  • MusicPlayer2

    • 💻GitHub 项目：zhongyang219/MusicPlayer2
    • ✨方便、易用，支持格式广的优秀本地播放器
    • 💖相比于繁杂的 Foobar2000，我更喜欢这个

  • MusicTag

    • 💻国内作者的 vinlxc 绿色软件：音乐标签pc版
    • 🎈音乐标签编辑器，支持 FLAC, APE, WAV, MP3 等音频格式

  • CUE Splitter

    • ⭕微软商店中的 UWP 版本
    • ✨简单且好用的 CUE 分轨工具

  • YesPlayMusic

    • 💻GitHub 项目：qier222/YesPlayMusic
    • 🎈一个好看的第三方开源网易云音乐客户端
    • ✨去除了社交功能，其他功能基本均能正常使用
    • ⭕支持 UnblockNeteaseMusic
    • ⭕由于基于 Electron 开发，所以不够轻量

视频播放

  • 恒星播放器
    • 🎈比较良心的国产播放器，但是包含少量广告
    • ✨功能全面，同时有比较丰富的插件可供使用
    • ⭕安装即用，还不用专门配置 madVR（这也是我不推荐 PotPlayer 的重要原因）
    • ⭕现在已经转为付费，同时有毒瘤行为

电子书

  • Calibre
    • 🎈电子书管理软件
    • ✨能够阅读、标注电子书，支持 epub、mobi、azw 等格式
    • ✨能够编辑制作电子书，支持不同格式转换

提高生产力

效率工具

  • Ditto

    • 🎈加强的剪切板工具
    • ⭕尽管 Windows10 下的 win + v 对剪切板来说已经足够好用，但是它存储的时长和类型都有很大限制
    • ⭕微软商店中的 UWP 版本
    • ✨和 Snipaste 配合会有奇效
    • 💖来自剪切板的备份可能能够拯救你的某次重大工作

  • Snipaste

    • 🎈加强的截图工具
    • ⭕尽管 Windows10 下的 Print Screen 对截图来说已经足够好用，但 Snipaste 更多的是方便了专门拿 QQ 截图的用户
    • ⭕微软商店中的 UWP 版本
    • ✨贴到屏幕上是我认为最有用的功能

  • Beyond Compare

    • 🎈强大的文件对比工具
    • ✨高效对比整个驱动器和文件夹，检查大小和修改时间，或者逐字节完整验证每个文件
    • ⭕能够帮助同步、合并文件夹
    • ⭕能够逐个像素对比图片

笔记文档

  • Anki

    • 🎈记忆卡片式的笔记工具
    • ⭕可以多平台同步数据
    • ✨有大量由他人整理好的牌组使用

  • Obsidian

    • 🎈具有强大功能的 Markdown 笔记
    • ⭕支持独特的“双向链接”与“关系图谱”
    • ✨有大量的插件

学习办公

  • Mathpix

    • 🎈对图片 OCR 输出 LaTeX 格式文本
    • ⭕免费用户每月仅能使用50次，通过邀请好友能够获得更多次数
    • ⭕可以通过免费申请其 OCR API 获得大量次数，然后利用 Github 上的第三方项目来进行使用

  • 知云文献翻译

    • 🎈便于 pdf 格式文献翻译的阅读器
    • ✨支持包括 Google、DeepL 在内的多种翻译引擎
    • ✨对 pdf 换行符、空格有很好的优化，会将原文及译文置于页面右侧
    • ⭕可以选择付费或者朋友圈宣传来获取更多翻译引擎的使用资格
