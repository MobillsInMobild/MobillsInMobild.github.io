多重背包问题

  • 问题：nnn 个物品，每个物品的体积是 viv_ivi​，价值是 wiw_iwi​，背包的容量是 mmm。若每个物品至多装 sis_isi​ 个，且不能超过背包容量，则背包的最大价值是多少？
  • 思路：多重背包问题不能像完全背包问题那样基于公式优化。它的优化思路是二进制优化——对于有 sis_isi​ 件的物品 iii，可以打包为 [log⁡si][\log s_i][logsi​] 个物品，每包有 1,2,4,⋯ ,2k,C1,2,4,\cdots,2^k,C1,2,4,⋯,2k,C 件物品 i′i'i′，其中 k=[log⁡si]−1k = [\log s_i] - 1k=[logsi​]−1。这样，就转换为了 01 背包问题。
    • 时间复杂度 O(nmlog⁡s)O(nm\log s)O(nmlogs)

分组背包问题

  • 问题：nnn 个物品，每个物品的体积是 viv_ivi​，价值是 wiw_iwi​，背包的容量是 mmm。若每组物品最多只能装一个，且不能超过背包容量，则背包的最大价值是多少？

区间 DP

  • 线性 DP 的扩展，以区间长度为阶段，使用能足够表示区间的坐标表示状态

  • 特点：对整个问题设最优值，枚举合并点，将问题分解为左右两个部分，最后合并两个部分的最优值得到原问题的最优值。

  • 模板

    • 所有的区间 DP 问题，第一维都是枚举区间长度，一般 len = 1 用来初始化，枚举从 len = 2 开始，第二维枚举起点 i （右端点 j 自动获得，j = i + len - 1）

数位统计 DP

  • 数位：把一个数字按照个、十、百、千等等逐位拆开，关注它每一位上的数字。
  • 数位 DP 的基本原理：对于位数比较多的数，这样的过程中有许多重复的部分。我们可以把这些过程归并起来，将这些过程中产生的计数答案也都存在一个通用的数组里。此数组根据题目具体要求设置状态，用递推或 DP 的方式进行状态转移。
    • 数位 DP 中通常会利用常规计数问题技巧，比如把一个区间内的答案拆成两部分相减（类似于前缀和）

计算问题

  • 问题：给定两个整数 aaa 和 bbb，求 aaa 和 bbb 之间的所有数字中 0∼90\sim 90∼9 的出现次数
  • 思路：实现 count(n,x)，表示 1∼n1\sim n1∼n 中，数字 xxx 出现的次数。
    • 考虑 xxx 在 n=abcdefgn=abcdefgn=abcdefg 第四位 ddd 时出现的次数。可以将 nnn 看成 yyyxzzzyyyxzzzyyyxzzz
      • 当 000≤yyy≤(abc−1)000\leq yyy \leq (abc - 1)000≤yyy≤(abc−1) 时，000≤zzz≤999000\leq zzz \leq 999000≤zzz≤999：
        • 当 x≠0x\neq 0x​=0，此时 xxx 出现了 abc×1000abc\times 1000abc×1000 次
        • 当 x=0x= 0x=0，由于 000000000 时 nnn 实际并没有第四位 xxx，所以此时 xxx 共出现了 (abc−1)×1000(abc - 1)\times 1000(abc−1)×1000 次
      • 当 yyy=abcyyy=abcyyy=abc 时：
        • 当 d<xd < xd<x，abcdefg<abcdefgabcdefg < abcdefgabcdefg<abcdefg，此时 xxx 出现 0 次；
        • 当 d=xd = xd=x，000≤zzz≤efg000\leq zzz \leq efg000≤zzz≤efg，此时 xxx 出现 efg+1efg+1efg+1 次
        • 当 d>xd > xd>x，000≤zzz≤999000\leq zzz \leq 999000≤zzz≤999，此时 xxx 出现 100010001000 次
