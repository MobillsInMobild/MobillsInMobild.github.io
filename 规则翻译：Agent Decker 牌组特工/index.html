介绍

Agent Decker 是一款基于任务的单人牌组构建桌游，玩家在游戏中通过面对障碍获得装备和技能。每一回合都会有警报响起，所以你必须选择你要干掉的人。你是要去拿酷炫的武器，还是要去干掉安全摄像头？

每个任务大约需要 20 分钟，游戏包含五个按顺序执行的任务。 还有一个高分系统，因此您可以尝试打破自己的最佳成绩，或与其他玩家比较您的分数！
